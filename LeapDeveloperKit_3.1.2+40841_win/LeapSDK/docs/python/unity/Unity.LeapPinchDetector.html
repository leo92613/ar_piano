LeapPinchDetector¶

Class description

class Leap::Unity::PinchUtility::LeapPinchDetector

A basic utility class to aid in creating pinch based actions.

Once linked with an IHandModel, it can be used to detect pinch gestures that the hand makes.

Property

bool DidChangeFromLastFrame

Returns whether or not the value of IsPinching is different than the value reported during the previous frame.

bool DidEndPinch

Returns whether or not the value of IsPinching changed to false between this frame and the previous.

bool DidStartPinch

Returns whether or not the value of IsPinching changed to true between this frame and the previous.

bool IsPinching

Returns whether or not the dectector is currently detecting a pinch.

float LastPinchTime

Returns the value of Time.time during the most recent pinch event.

float LastUnpinchTime

Returns the value of Time.time during the most recent unpinch event.

Vector3 Position

Returns the position value of the detected pinch.

If a pinch is not currently being detected, returns the most recent pinch position value.

Quaternion Rotation

Returns the rotation value of the detected pinch.

If a pinch is not currently being detected, returns the most recent pinch rotation value.
