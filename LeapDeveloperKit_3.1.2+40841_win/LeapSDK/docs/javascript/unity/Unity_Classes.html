Leap.Unity Namespace¶

The Leap Motion core assets include scripts defining the following C# classes:

Core Classes  
LeapHandController The central class that takes tracking data and drives the hands in a scene.
LeapProvider An Abstract class that defines the functions necessary to provide frames of tracking data.
LeapServiceProvider Implements |LeapProvider|_ to provide tracking data from the Leap Motion service.
LeapImageRetriever Converts images from the Leap Motion cameras into Unity textures.
HandFactory An abstract class that defines the functions necessary to represent hands.
HandPool Implements |HandFactory| to build |HandRepresentation|_ objects from the models in its model collection and tracked hands.
HandRepresentation An abstract class that defines the properties of a hand in the scene.
HandProxy Implements |HandRepresentation| to combine a Leap Hand object with a set of physics or graphics models.
Hand Models  
IHandModel Defines the interface used to access hand model objects by the rest of the system. Implement this interface to create entirely different types of hands.
HandModel The base class for most of the hands in the asset packages. Works with the |FingerModel|_ base class.
DebugHand A utility graphics hand model that draws lines for the bones of a hand. Uses the Unity Debug.DrawLine() function. These hands are only visible in the scene view.
RiggedHand A graphics hand class that sets the rotations and, optionally, the positions of the rigged bones in a mesh model.
RigidHand (Physics) A physics hand class that moves the rigidbodies and colliders to match the tracked hand in a manner appropriate for the Unity physics engine.
SkeletalHand A graphics hand class that moves the transforms attached to game objects representing each part of the hand.
CapsuleHand A grahics hand class that draws a stylized representation of a hand.
Hand Behaviors  
HandTransitionBehavior An abstract base class defining the functions called when tracking for a hand is initiated or lost.
HandDrop A transition behavior that interpolates a hand to and from its starting position in the scene when a hand transition occurs.
HandEnableDisable A behavior that activates or deactivates the hand object when a hand transition occurs.
Finger Models  
FingerModel The base class for most finger models in the assets. Works with the |HandModel|_ class.
RiggedFinger Extends |SkeletalFinger|_ to drive the bones in an animated mesh with the bone data in the tracked finger.
RigidFinger (Physics) Extends |SkeletalFinger| to move the rigidbodies and colliders in a physics hand.
SkeletalFinger Extends |FingerModel|_ to move game object transforms according to the bones in the tracked finger.
Utility Scripts  
LeapDeviceInfo A struct that describes the attibutes of a Leap Motion hardware device.
DisconnectionNotice Fades an image in and out of view when a connection to the Leap Motion service cannot be established or is lost.
FrameRateControls Adjusts the Unity frame rate via the keyboard for testing.
LeapUnityExtensions Provides extensions for converting between related Leap Motion and Unity classes.
HandFader Fades a hand based on its confidence value.
TestHandFactory Creates a “dummy” frame containing a right and a left hand.
Utils A place for utility functions. Currently contains |Utils.IgnoreCollisions|_.
VR Scripts  
EyeType Classifies which eye the Unity camera is rendering.
LeapVRCameraControl Manages camera and rendering shader parameters.
LeapVRTemporalWarping Interpolates shader transform paramter to synchronize the tracking data with the camera images.
